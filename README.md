## Installation:
```
git clone https://github.com/technicca/foofetch
cd foofetch
sudo chmod +x run.sh
./run.sh
foofetch
```

or

```
git clone https://github.com/technicca/foofetch
cd foofetch
make all
make install
foofetch
```

-reset to clear cache
